1. Double click Keymagic1.1.exe
2. Follow the installer wizard and finish the installation.
3. Double click Keymagic icon on desktop to start keyboard manager.
4. At taskbar (near clock display), right click KeyMagic to launch keyboard manager.
5. At left panel, Add New button to add your keyboard files. Under KeyBoards folder. (*.kmk)
6. If you check "use from this path", application will use keyfile from that path, by then
   you will need to provide full path including drive letter.
7. If you want to generate your custom keyboard, use Keymap Generator(KeyMapper.exe).
8. KeyMagic, keylayout script files are under KeyMapper folder. (*.kms)
9. License and other info, refer doc folder.
